EXPLANATION

moviesnack

This mobile project was developed using the Flutter framework and the Dart programming language. It allows users to add movies by entering details such as the movie title, trailer link, poster image, director, producer, and more. Users can also rate each movie. All the data is stored in a local database hosted on my laptop, and the app communicates with it using a custom-built API. This setup demonstrates how the app handles data input, rating functionality, and backend integration in a structured and efficient way
